## Getting Started
1. ./client/.env.localを作成する
```
# .env.local
HOTPAPER_API_KEY=xxxxxxx
```
2. ディレクトリ移動
```
cd ./client
```
3. ライブラリのインストール
```
npm install
```
4. サーバー起動
```
npm run dev
```
